## Troubleshooting

### Page doesn't load
- Check file path
- Refresh browser
- Clear cache
- Try different browser

### Images don't show
- Check image format (PNG/JPG)
- Check image size
- Try different image
- Check browser console

### Touch not working
- Use real touch device (not trackpad)
- Check viewport width < 768px
- Try refreshing
- Check mobile browser

### Buttons not responsive
- Try actual mobile device
- Check browser zoom is 100%
- Try different browser
- Check inspector (F12)
